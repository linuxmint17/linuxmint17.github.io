C++ 11/14/17/20 new feature

Created2020-05-24|Updated2020-05-24
|Post Views:

自从C++11以来的新特性

  • auto 自动类型推导
  • constexpr
  • nullptr 关键字, 用于取代 NULL, 代表空指针
  • for 循环中的 语法糖 for ( a : array) { }
  • lambda 表达式， 也就是匿名函数

自从C99以来的新特性

  • 变长数组 VLA;
  • 结构体 按成员初始化
  • restrict 关键字
  • bool 类型
